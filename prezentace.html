What's done

The game has gone through 9 milestones — from core mechanics to full 3D visuals and brand identity. Currently in the beta cycle (v74) and final polishing.

Gameplay & mechanics

  • Core loop — carriers, belt, cannon, score by color match, honeycomb carrier activation.
  • Adaptive layout — 4–7 carrier rows, responsive sizing 38–54 px, iOS safe area, touch fix.
  • Funnel physics — multi-segment bezier collider, sliding contact, smoothed normals, anti-escape.
  • Blocks, walls, mystery carriers — reveal animation, cascade pop, denial shake; mystery with wipe reveal (balls under the cover) — v74.60.
  • End-screen "Next →" — smooth continuation through the LEVELS sequence after finishing a level, cycles around — v74.58.

2.5D visuals (M6–M9)

  • Three.js scene — stylized 3D rendering (toon + lambert + matcap); pixels, blocks, projectiles, carriers, belt.
  • Unified 3D grid — walls as a monolith (BFS + ExtrudeGeometry), rounded silhouette, outline.
  • Bottom unified frame — belt + gap + arena as one shape with a "punched-out" hole, stencil clipping.
  • 3D gun + muzzle flash + smoke, cartoon squash&stretch, motion trails, BG vignette, sparkles.
  • Block outline silhouette — outline follows the block's overall shape (L/T/+) instead of per-cell strips — v74.56.
  • HP text in 3D blocks — Impact font, depth shadow, -30 % size — v74.55/57.

Audio palette (v74.9–v74.61)

  • SFX — pop, bounce, 4-var shoot, step (cannon move), suck (hole), carrier click, denial.
  • Pentatonic pitch quantization — action sequences sound like a melodic phrase, not random beeping.
  • Adaptive music — 5 layers (kick / hihat / clap / pad / belt tick) with build-up and decay after idle.
  • Mix — pop fatigue, throttle, master volume balance tuning (shoot under pop, restrained cannon).

Editor, AI, infra

  • Level editor — photo import, block editor, multi-variant layouts; card UI with mini thumbnails and a 2-row layout — v74.57.
  • In-game filter + share link — checkbox per level (unchecked levels stay editor-only); 🔗 button copies the GH Pages URL to clipboard — v74.57.
  • Theme picker per level — a default theme per level, optionally overriding the global choice — v74.56.
  • Photo color minimizer — preview mode with threshold, live preview, undo; trims the palette on photo import — v74.62.
  • AI tester — solver hierarchy, difficulty score 0–100, heat map, curve editor + auto-tune SA.
  • Brand "Plop!" — comic icon, version badge, PWA manifest, Bangers font in the brand identity.
  • PWA service worker — network-first, auto-reload on update, offline fallback, safe-area inset.

What's in progress

Beta cycle (v74) — feature complete, but we're still wrapping up these items before release.

Beta polish — P0 / P1

  • Blocks fix (P0) — block mechanics need a re-check after the 3D switch; silhouette outline done (v74.56), concave corners see P2.
  • Ball‑to‑hole check immediately (P1) — right after loading instead of waiting for the end of rounds.
  • FPS drain diagnosis (P1) — the game burns frames even idle, hunting the cause.

Beta polish — P2 details

  • Speed up the belt and flow speed‑up after carriers empty (≈ 2× speed until end of level).
  • Ball highlights for dark colors (rim‑light) — black and dark blue are hard to see in the grid.
  • Block outline concave corners — L/T/+ have a visual artifact, clean fix via merged geometry.
